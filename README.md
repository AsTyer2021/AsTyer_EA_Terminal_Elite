　　第二幕：战场上的将军——交易逻辑与信号共振，像“武林外传”的同福客栈大乱斗

　　OnTick()函数像“武林外传”里的佟湘玉，掌柜的每秒巡视店堂，支持8品种（EURUSD到NZDUSD，像“葫芦七兄弟”各司其职，第八个自定义如“野葫芦”）。策略是马丁格尔：首单像“赌神”高进的底牌，基于RSI、CCI、MACD“共振”开仓——这仨像“桃园三结义”的刘关张：RSI加成2倍如刘备的仁义，CCI主导如关羽的武力，MACD确认如张飞的吼声；齐心就sig=1（多）或-1（空），不同心就sig=0，中性如“和尚摸秃驴——没戏”。亏了加仓，拉间距（倍率1.6，如“兔子拉车——越拉越远”），手数翻倍（1.2，但动态curM基于CCI，限1-3，如“孙悟空七十二变——小变大，但别变妖怪”）。

　　升级马丁在CheckAndExecuteExtraOrder有盈利单（反转如“咸鱼翻身”），价格超间距，就额外下单，手数同最后单，还设保本SL——这像“以毒攻毒”的中医偏方：亏成狗，反转时猛补一针，赌它起死回生。比喻海阔天空：市场如“周润发的神之左手”，一巴掌扇飞本金；你这EA如“叶问的咏春拳”——寸劲加仓，但遇上“泰森”就成“挨打的沙包”。手机遥控用0.01手挂单，尾数22开、99关，像“间谍过家家”的暗号：“燕子穿云，帽子在左手——开多！否则关空，像关灯省电”。

　　第三幕：守护神与风险控制——止损止盈与自动优化，像“西游记”的紧箍咒

　　风险如“紧箍咒”：总浮亏超限（金额or比例）就全平，像唐僧念咒，孙猴子抱头鼠窜；独立止损如“猪八戒的耙子”——亏到点，品种平仓，干净利落。点差超35不开，如“守株待兔”的兔子太远，不追；停止时段如“新闻联播”的广告时间，避开高峰。保本止损：盈利单，设SL=平均价+(tp/10)，棘轮如“电梯只上不下”——盈利上移SL，亏了就“电梯直达地狱”。

　　自动优化每4小时用H4 ATR（12根，如“十二生肖”轮值）算波动，tp=ATR0.191（至少40，如“小步快跑”），step=ATR1.618（至少120，黄金分割如斐波那契的“兔子繁殖”——越生越多，但别生出怪物）。这如孙子兵法的“知彼知己”：知市场（ATR如敌情），知参数（优化如兵法），百战不殆——但外汇如“赌局”，你知彼彼不知你，结果“百战百败，家底败光”。平仓确认弹窗如Windows的“删除文件？”——“您确定要归零清仓吗？单数如葫芦娃，手数如西瓜——一刀切，汁水四溅”。
